# worth the weight 

cute grade calculator website ^^ 

## description 

this website takes in inputs of current grade and final weight. it calculates: minimum final exam score needed to get desired course grade, final course grade after taking final exam, and gpa impact/new gpa after taking a class. the website also features a interactive visualizer showing max - min final grades possible, with a slider, tool tip, and line graph. 

## execution 

this is a github-hosted website, so just access through the link in the about section. cheers!
